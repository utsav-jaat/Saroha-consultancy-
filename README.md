## Project Overview

Saroha Consultancy is a recruitment and staffing solutions website built with pure HTML and CSS. The site connects job seekers with employment opportunities across various sectors including airport jobs, data entry, security, and more.

---

https://github.com/utsav-jaat/Saroha-consultancy-/blame/main/index.html#L215

**Key Sections:**
- **Header:** Navigation menu with logo and links (Home, Services, Jobs, Contact)
- **Hero Section:** Main call-to-action with "Connecting Talent With Opportunity" tagline
- **Services Section:** 4 service cards (Recruitment, Staffing, Permanent Hiring, Job Placement)
- **Featured Jobs:** 3 job listings (Data Entry Operator, Security Guard, Office Assistant)
- **Contact Section:** Phone number (7838005192) and email (sarohaconsultancy@gmail.com)
- **WhatsApp Widget:** Fixed floating button for WhatsApp contact

**Styling Features:**
- Responsive grid layout with `auto-fit` and `minmax()`
- Dark header (#0f172a) with white text
- Card-based design with shadows
- Green apply buttons (#16a34a)
- Blue primary buttons (#2563eb)
- Fixed WhatsApp button (#25D366)
